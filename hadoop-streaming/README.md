# hadoop-streaming
Напишите программу, которая реализует mapper и reducer для задачи WordCount в Hadoop Streaming.

Sample Input:

```
Vivere est cogitare
Vivere militate est
Scientia potentia est
```

Sample Output:

```
Scientia	1
Vivere	2
cogitare	1
est	3
militate	1
potentia	1
```

## Запуск
```
sh sun.sh
```
