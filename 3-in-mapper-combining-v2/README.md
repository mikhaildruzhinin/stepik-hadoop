# in-mapper-combining-2
Напишите программу, которая реализует In-mapper combining v.2 для задачи WordCount в Hadoop Streaming.

Sample Input:

```
aut Caesar aut nihil
aut aut
de mortuis aut bene aut nihil
```

Sample Output:

```
aut	6
mortuis	1
bene	1
Caesar	1
de	1
nihil	2
```

## Запуск
```
sh run.sh
```
