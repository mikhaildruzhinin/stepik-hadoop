# distinct-values-v2
Реализуйте reducer из задачи Distinct Values v2.

Reducer принимает на вход строки, каждая из которых состоит из разделенных табуляцией значения и названия группы.

Sample Input:

```
1	a
1	b
1	b
2	a
2	d
2	e
3	a
3	b
```

Sample Output:

```
a	3
d	1
b	2
e	1
```

## Запуск
```
sh run.sh
```
